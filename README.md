# Sales-Revenue-Analysis-Dashboard-using-Power-BI

## 📌 Project Overview
This project is a Sales & Revenue Analysis Dashboard built using Power BI.  
The dashboard helps analyze sales performance, revenue trends, profit distribution, and top-performing products using interactive visualizations and slicers.

The goal of this project is to gain practical experience in:
- Data visualization
- KPI tracking
- Business insight generation
- Interactive dashboard creation

---

## 📂 Dataset Used
Dataset: Superstore Sales Dataset

The dataset contains:
- Order details
- Customer information
- Product categories
- Sales and profit data
- Regional sales information

Columns used in analysis:
- Order Date
- Region
- Category
- Sub-Category
- Product Name
- Sales
- Profit
- Quantity
- Segment

---

## 🛠️ Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders

### Visualizations
- Sales Trend Analysis
- Top Selling Products
- Regional Sales Distribution
- Profit Analysis by Category

### Interactive Features
- Region Slicer
- Category Slicer
- Segment Slicer
- Date Filter
